About Us

Project Purpose

This project aims to provide a web-based tool for predicting the risk of heart attack based on user-provided health and lifestyle data. Our goal is to raise awareness and provide valuable insights that can help users make informed decisions about their health.

The Team

Our team consists of dedicated data scientists and software developers who are passionate about healthcare and technology. We worked collaboratively to design and develop this application, ensuring it is user-friendly and accurate.

Technologies Used

The application is built using Flask for the web framework, scikit-learn for the machine learning model, and HTML/CSS for the frontend design. We utilized various libraries and tools to ensure robust performance and a seamless user experience.

Acknowledgments

We would like to thank the open-source community for providing valuable libraries and tools that helped in the development of this project. Special thanks to the data sources and contributors who made this project possible.
