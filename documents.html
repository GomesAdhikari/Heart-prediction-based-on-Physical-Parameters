Project Overview

This project aims to predict the risk of heart attack based on various health and lifestyle factors. The web application allows users to input their details and receive a prediction about their risk of heart attack.

Model Details

The model used for prediction is a Support Vector Machine (SVM). It was trained using a dataset of health and lifestyle features. The model's performance was evaluated using cross-validation techniques to ensure its reliability.

Data Description

The dataset includes features such as Age, Sex, Cholesterol, Smoking, Obesity, Exercise Hours Per Week, Stress Level, Sedentary Hours Per Day, Income, Triglycerides, Physical Activity Days Per Week, Systolic, and Diastolic. The data was preprocessed to handle missing values and normalize the features.

Methodology

Feature engineering was performed to select the most relevant features for prediction. The SVM model was chosen based on its ability to handle high-dimensional data and provide accurate predictions. The model's performance was assessed using metrics such as accuracy, precision, recall, and F1-score.

User Guide

To use the application, enter your details in the form on the home page and submit. The result will be displayed on a separate page, including a risk prediction and suggestions based on your inputs.

Future Work

Future improvements may include expanding the dataset, experimenting with different models, and incorporating additional features to enhance the prediction accuracy. User feedback will be valuable for further refinement of the application.
